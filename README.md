### About Me

I'm a software engineer who focuses on building complete systems — not just frontends or backends, but the whole thing. I design, architect, and ship full-stack applications that handle real business operations: point-of-sale terminals, installment tracking, inventory management, financial accounting, and multi-tenant cloud platforms.

I care about clean architecture, type safety, and building software that actually works under real-world conditions.

---

### What I'm Working On

- **Retail Matrix** — A full-featured ERP/POS system built with Electron + React (desktop) and a .NET cloud backend. Handles sales, inventory, installments, debt tracking, financial accounting, multi-currency, thermal printing, and subscription-based feature gating across organizations.

- **Cloud Platform** — Multi-tenant SaaS infrastructure with organization management, subscription plans, role-based access control, and feature toggles.
